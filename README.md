# 🏗️ Cpp-Machines Suite v1.0

Bienvenido a **Cpp-Machines**, un ecosistema modular de herramientas avanzadas desarrolladas en **C++**. Este proyecto demuestra la implementación de soluciones de bajo nivel para desafíos complejos de sistemas, comunicaciones y seguridad.

---

## 🛠️ Las Máquinas de la Suite

Cada módulo ha sido diseñado para funcionar de forma independiente o como parte de un sistema distribuido:

| ID | Nombre | Especialidad | Tecnologías Clave |
| :--- | :--- | :--- | :--- |
| **M-01** | **Deep Memory Scanner** | Manipulación de Memoria | AOB (Array of Bytes), ASLR Bypass |
| **M-02** | **Neural Node Controller** | Comunicaciones | Red Asíncrona, Boost.Asio |
| **M-03** | **Cluster Orchestrator** | Orquestación | Balanceo de carga, Master/Worker |
| **M-04** | **Hydra-Net Shield** | Ciberseguridad | Proxy, Firewall de paquetes, Sandbox |

---

## 🚀 Inicio Rápido

### 1. Requisitos de Software
Para compilar y ejecutar estas máquinas, necesitas el entorno de desarrollo oficial:
* **IDE:** [Visual Studio 2022 Community](https://visualstudio.microsoft.com/es/vs/community/)
* **Compilador:** MSVC (x64 recomendado)
* **Librerías:** [Boost C++ Libraries](https://www.boost.org/) (Necesario para M-02)

### 2. Instalación
1. Clona el repositorio:
   git clone https://github.com/TU_USUARIO/Cpp-Machines.git
2. Abre la solución en Visual Studio.
3. Configura el proyecto en modo **Release x64**.
4. Ejecuta como **Administrador** (necesario para M-01).

## 🔍 Filosofía de Desarrollo
- **Modern C++:** Uso de estándares C++17/20, lambdas y gestión de memoria inteligente (`std::shared_ptr`).
- **Seguridad por Diseño:** Implementación de capas de firewall y entornos aislados (Sandbox).
- **Eficiencia:** Algoritmos de balanceo de carga para optimizar el uso de la CPU.
