# 🏗️ Cpp-Machines Suite v1.0

Bienvenido a **Cpp-Machines**, una colección de herramientas de alto nivel 
desarrolladas en C++ para demostrar el poder del lenguaje en entornos de 
sistemas, redes y seguridad.

Esta suite está diseñada como un ecosistema modular donde cada "máquina" 
resuelve un desafío técnico específico.

## 🛠️ Las Máquinas de la Suite

| Máquina | Nombre | Especialidad |
| :--- | :--- | :--- |
| **M-01** | Deep Memory Scanner | Localización de patrones AOB y manipulación de memoria. |
| **M-02** | Neural Node Controller | Servidor de red asíncrono basado en Boost.Asio. |
| **M-03** | Cluster Orchestrator | Gestión de nodos trabajadores y balanceo de carga. |
| **M-04** | Hydra-Net Shield | Proxy inteligente y Firewall de filtrado de paquetes. |

## 🚀 Filosofía del Proyecto
Cada módulo ha sido programado bajo tres pilares fundamentales:
1. **Eficiencia:** Uso de C++ moderno (C++17/20) y Smart Pointers.
2. **Seguridad:** Implementación de Sandboxes y capas de validación.
3. **Escalabilidad:** Arquitecturas preparadas para crecer y conectarse entre sí.

## 📦 Requisitos Generales
- IDE: Visual Studio 2022 / GCC.
- Arquitectura: x64 recomendada.
- Dependencias: Boost Libraries (requerido para M-02).
